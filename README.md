Welcome to my portfolio! Built with React, Tailwind CSS, and Framer Motion, this single-page application is designed to showcase my academic foundation, software projects, engineering skills, and professional experience.

🚀 Live Site: anantadev.netlify.app

🛠️ Built With

Core Framework: React.js + Vite (for ultra-fast bundling and HMR)
Styling: Tailwind CSS (modern utility-first layout structuring)
Animations: Framer Motion (smooth on-scroll triggers and entry curves)
Icons: Lucide React (consistent vector visuals)
Forms Processor: Web3Forms API (secure, serverless form handler)

🚀 Getting Started Locally

To run this project on your machine, follow these simple setup steps:

1. Clone the repository

git clone https://github.com/Ananta0902/portfolio.git
cd portfolio/portfolio-website


2. Install dependencies

npm install


3. Activate the Contact Form

Generate a free access key at Web3Forms.

Open src/sections/Contact.jsx.

Locate WEB3FORMS_ACCESS_KEY and replace "YOUR_ACCESS_KEY_HERE" with your real key.

4. Run the development server

npm run dev


Open your browser to http://localhost:5173 to view your local changes.

5. Compile the production bundle

npm run build


This builds and compiles optimized static HTML, CSS, and JS files directly inside the dist/ directory, ready for production deployment.
